# SegurEntry

Repositorio para el proyecto **SegurEntry** (backend Django + frontend Angular).

## Estructura
- `backend/`: API y lógica de servidor.
- `frontend/`: aplicación web Angular.

## Git
Este repositorio incluye `.gitignore` para evitar subir artefactos generados (dependencias, builds), archivos de configuración local y secretos.

## Notas
Si necesitas añadir claves o configuración local, crea archivos basados en los ejemplos del proyecto (p. ej. `.env`) y asegúrate de no commitearlos.
